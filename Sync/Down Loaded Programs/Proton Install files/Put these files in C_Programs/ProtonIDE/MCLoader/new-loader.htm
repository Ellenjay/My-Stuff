Loader Update Information

Version 3.2.0.5

  • Changed the way the loader detects COM ports. A search is made in the system registry under HKEY_LOCAL_MACHINE\hardware\devicemap\serialcomm. This changed was required in order to support the increasing number of virtual COM port drivers. Please note that a port shown as available in the registry does not mean it is available for use by the loader.
  • Problem with erasing 16F88 parts, this should now work OK.
