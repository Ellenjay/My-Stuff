Changing Default Device Name and XTAL

You can now get the IDE to use a default device name and crystal setting for your project. These values will be passed to the compiler, unless overridden in your main program. For example, the default device and crystal settings are 16F84 and 4MHz. If you want you defaults to be 18F452 and 20MHz, then add the following entries to 'user.mcd' under the DATABASE keyword

Core = dc16
Device = "18F452"
OSC = 20

You could of course just modify 'database.mcd', but this will be overwritten every time you perform an online update.
