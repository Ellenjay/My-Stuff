TeamViewer Portable
===================

ACHTUNG: Die Benutzung von TeamViewer 5 Portable benötigt eine TeamViewer Premium
	 oder Corporate Lizenz.

Dies ist ein vollständiges TeamViewer Supporter Modul, das direkt ohne Installation gestartet werden kann
(z.B. von einem USB-Stick). Die Einstellungen werden nicht in die Registry gespeichert.

Bitte editieren Sie die Datei TeamViewer.ini und ersetzen Sie den Platzhalter-Lizenzcode in der Kategorie
'License' mit Ihrer Premium / Corporate Lizenz.

Mit der Einstellung writeconnectionlog=1 in der Datei TeamViewer.ini können Sie erzwingen, dass TeamViewer
eine log-Datei mit detaillierten Verbindungsinformationen schreibt.

Mit der Einstellung importsettings=1 können Sie den automatischen Einstellungsimport aktivieren.
TeamViewer wird alle Einstellungen aus einer Datei namens tv.ini importieren, welche im gleichen Verzeichnis 
wie die TeamViewer.exe platziert sein muss. Sie können die tv.ini erstellen, indem Sie Ihre Einstellungen 
in den erweiterten Optionen von TeamViewer exportieren.

Wenn Sie TeamViewer Portable unter Windows 98 nutzen wollen, müssen Sie die Datei
unicows.dll, welche von http://www.teamviewer.com/download/unicows.dll heruntergeladen werden kann, 
dem Verzeichnis der TeamViewer.exe hinzufügen.

Bei der Nutzung von TeamViewer Manager Portable, müssen Sie diesen direkt aus der installierten Version
von TeamViewer Manager über Extras exportieren. Wichtig ist, dass sich dieses Verzeichnis und das Verzeichnis 
von TeamViewer Portable auf gleicher Verzeichnisebene befinden.
